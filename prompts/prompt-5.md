# Prompt
Implemente o aplicação web utilizando Python e Flask com as seguintes funcionalidades:

- Autenticação de usuários com login e senha
- Diferenciação de perfis (usuário comum e administrador)
- Upload de arquivos pelos usuários autenticados
- Listagem e download dos arquivos enviados
- Administradores podem visualizar arquivos de todos os usuários
- Usuários comuns só podem acessar seus próprios arquivos
- Utilize sessões para controle de autenticação
- Armazene os arquivos no servidor
- Utilize SQLite para persistência de dados
- A aplicação deve ser funcional e simples, priorizando clareza do código

Não é necessário foco em frontend.
